Note Taking app created wtih C# .Net Maui which follows MVVM architecture pattern. <br>

Features: <br>
Horrific UI <br>
Does not use CommunityToolkit.MVVM source generators <br>
Has an SQLite db connected using sql-pcl-net <br>
